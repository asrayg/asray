just finished everything for my senior design (capstone) project. it was a collaboration with collins aerospace.

we built a project management system for handling artifacts across different model-based design tools. engineers currently rely on memory to keep track of how artifacts from different systems connect. our solution formalizes those relationships and remembers them automatically. we also implemented graph optimizations and improved rendering performance on both the backend and frontend.

it was great working with Arjun Patel, Issmale Bekri, and Akhil Pallem. thanks Scott Meyers for all the support!
